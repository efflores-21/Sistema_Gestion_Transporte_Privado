Sistema de Gestión de Transporte Privado
Este proyecto implementa una aplicación para administrar un servicio de transporte privado utilizando Java 17, Maven y OpenXava 7.6.1. El sistema fue generado mediante el archetype oficial de OpenXava y organiza su estructura alrededor de un modelo de datos que incluye clientes, conductores, vehículos, reservas, viajes, facturas, pagos, proveedores, repuestos y mantenimientos.
La arquitectura se basa en entidades JPA, donde las clases principales (Cliente, Conductor y Vehículo) extienden una clase base con identificadores UUID. Los tipos auxiliares, como TipoCliente, TipoLicencia y TipoVehiculo, se mantienen como entidades simples que no generan módulos propios, ya que se integran directamente en los formularios mediante @DescriptionsList o vistas reducidas. Las relaciones entre entidades siguen fielmente el diagrama ERD, permitiendo manejar procesos como reservas que se transforman en viajes, viajes que generan facturas y facturas asociadas a pagos.
El proyecto puede ejecutarse fácilmente con Jetty mediante Maven y está preparado para funcionar con H2 por defecto, aunque puede configurarse para otros motores SQL. Su código está organizado de manera clara para facilitar extensiones posteriores, como validaciones adicionales, seguridad por roles o acciones personalizadas dentro de OpenXava.
En resumen, este sistema sirve como una base sólida y modular para gestionar de forma completa la operación de transporte privado, manteniendo una estructura ordenada, relaciones coherentes y una interfaz generada automáticamente por OpenXava.
